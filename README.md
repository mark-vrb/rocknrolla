1. Run the app:
```
$ python manage.py runserver
```

2. Create an admin user:
```
$ python manage.py createsuperuser
```

3. Make migration scripts:
```
$ python manage.py makemigrations [appname]
```

4. Perform migration:
```
$ python manage.py migrate
```

5. Play with the API
```
$ python manage.py shell
```
